# 憨吉椪
過年初四爸爸媽媽帶我去旗津玩，剛好有機會吃超有名的「椪嫂蕃薯椪」。當時不小心買太多顆，蕃薯椪又熱熱的不能放到明天，所以我當天連吃了四顆。我一邊吃著蕃薯椪，一邊拿我的相機到出拍。我覺得我當天運氣很好，拍到好多酷酷的照片，蕃薯椪也甜甜的超級好吃，所以我很開心。
